# HIERACLES-NIX

## Overview

This repository aims to test and illustrate different ways to setup Hieracles (ruby gem) on NixOS.

- [Hieracles 0.4.2 with pre-generated files](./hieracles-0.4.2/README.md)
- [Hieracles 0.4.2 using bundix](./hieracles-0.4.2-bundix/README.md)
- [Hieracles 0.4.2 with manual patch](./hieracles-0.4.2-patched/README.md)
- [Hieracles 0.4.3 in a shell](./hieracles-0.4.3-shell/README.md)
